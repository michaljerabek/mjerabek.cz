Co vám mohu nabídnout?

Kóduji grafické návrhy webů do HTML, CSS a JavaScriptu.

  • HTML

    Správně označený význam obsahu podle nejnovějšího standardu HTML5.1. Přehledný, sémantický a přístupný kód.

  • CSS

    Stabilní layout a vzhled odpovídající návrhu. Ručně psané CSS a metodologický přístup pak zajistí udržovatelný kód.

  • JS

    Interakivní UI — galerie, kalendář, kalkulačka nebo třeba animace — zajistí čistý, efektivní, modulární a udržovatelný JS.

Potřebujete-li nejdříve vytvořit grafický návrh webu, můžu jej zajistit u svého bratra :·).

Pro zajištění funkcí jako odeslání formuláře budete potřebovat back-endového (PHP) vývojáře.
